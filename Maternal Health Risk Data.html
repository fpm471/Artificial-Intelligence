Normalización de datos¶

Algunos algoritmos (como las SVMs) sufren cuando los rangos de valores entre variables difieren significativamente. Es decir, que una variable se mueva en el intervalo [10,20] y otra en el intervalo [1000, 2000], har que el algoritmo se pierda.

Para evitar esto, una buena prctica es normalizar todos los valores, haciendo que estn siempre en un intervalo concreto, de esta manera evitamos estos posibles efectos negativos.

El cdigo que aparece a continuacin sirve para normalizar los datos utilizando la tcnica de StandardScaler, la cual transforma las variables para que tengan media cero y varianza unitaria. Asimismo, podemos observar que la normalizacin de los datos es til para evitar que algunas variables tengan ms peso en el modelo debido a su escala, y tambin para mejorar la estabilidad y convergencia de algunos algoritmos de aprendizaje automtico.

La SVM Lineal tiene un parmetro C:

  • Cmo vara el acierto (score) cuando aumentamos C?

El parmetro C en una SVM Lineal controla el grado de error que se permite en la clasificacin de los datos de entrenamiento. Un valor pequeo de C permitir un mayor error de clasificacin en el conjunto de entrenamiento, mientras que un valor grande de C intentar clasificar cada punto correctamente en el conjunto de entrenamiento.

  • Qu sucede cuando disminuye?

Cuando disminuimos el valor de C, el modelo puede tener una mayor capacidad de generalizacin y puede tener un mejor rendimiento en el conjunto de prueba, pero puede que tenga un bajo rendimiento en el conjunto de entrenamiento.

  • En qu escala se aprecian los cambios: Escala logartmica o escala lineal?

Se aprecian mejor los cambios en una escala logartmica que en una escala lineal, ya que el valor de C se multiplica por una constante en cada iteracin.

from sklearn.svm import SVC
from scipy.stats import uniform

# Definir una grilla de parmetros

# Crear un modelo SVM

# Realizar una bsqueda aleatoria en la grilla de parmetros utilizando RandomizedSearchCV

# Verificar el rendimiento del modelo utilizando los datos de prueba

import numpy as np
from sklearn import svm, tree, metrics
from sklearn.model_selection import train_test_split
from sklearn.datasets import load_wine

# carga los datos
data = load_wine()
X = data.data
y = data.target

# elegir la clase 1 como positiva y el resto como negativa
y_bin = np.zeros_like(y)
y_bin[y == 1] = 1

# divide los datos en conjuntos de entrenamiento y prueba
X_train, X_test, y_train, y_test = train_test_split(X, y_bin, test_size=0.3)

# entrena los modelos SVM y rbol de decisin
svm_model = svm.SVC(probability=True)
svm_model.fit(X_train, y_train)

tree_model = tree.DecisionTreeClassifier()
tree_model.fit(X_train, y_train)

# obtiene las probabilidades de la clase positiva para cada modelo
svm_probs = svm_model.predict_proba(X_test)
tree_probs = tree_model.predict_proba(X_test)

# calcula las mtricas de evaluacin para cada modelo
svm_preds = np.argmax(svm_probs, axis=1)
tree_preds = np.argmax(tree_probs, axis=1)

svm_cm = metrics.confusion_matrix(y_test, svm_preds)
tree_cm = metrics.confusion_matrix(y_test, tree_preds)

svm_report = metrics.classification_report(y_test, svm_preds)
tree_report = metrics.classification_report(y_test, tree_preds)

svm_f1 = metrics.f1_score(y_test, svm_preds)
tree_f1 = metrics.f1_score(y_test, tree_preds)

svm_precision = metrics.precision_score(y_test, svm_preds)
tree_precision = metrics.precision_score(y_test, tree_preds)

svm_recall = metrics.recall_score(y_test, svm_preds)
tree_recall = metrics.recall_score(y_test, tree_preds)

from sklearn.datasets import load_wine
from sklearn.model_selection import train_test_split
from sklearn.preprocessing import StandardScaler, MinMaxScaler, Normalizer
from sklearn.svm import SVC
from sklearn.metrics import accuracy_score

# Cargar el conjunto de datos de vinos
wine = load_wine()

# Dividir los datos en conjuntos de entrenamiento y prueba
X_train, X_test, y_train, y_test = train_test_split(wine.data, wine.target, test_size=0.2, random_state=42)

# Definir los normalizadores
scaler_zscore = StandardScaler()
scaler_minmax = MinMaxScaler()
scaler_l1 = Normalizer(norm='l1')
scaler_l2 = Normalizer(norm='l2')

# Entrenar un modelo SVM para cada normalizacin
svm_zscore = SVC(kernel='rbf', gamma='scale')
svm_minmax = SVC(kernel='rbf', gamma='scale')
svm_l1 = SVC(kernel='rbf', gamma='scale')
svm_l2 = SVC(kernel='rbf', gamma='scale')

X_train_zscore = scaler_zscore.fit_transform(X_train)
X_train_minmax = scaler_minmax.fit_transform(X_train)
X_train_l1 = scaler_l1.fit_transform(X_train)
X_train_l2 = scaler_l2.fit_transform(X_train)

svm_zscore.fit(X_train_zscore, y_train)
svm_minmax.fit(X_train_minmax, y_train)
svm_l1.fit(X_train_l1, y_train)
svm_l2.fit(X_train_l2, y_train)
